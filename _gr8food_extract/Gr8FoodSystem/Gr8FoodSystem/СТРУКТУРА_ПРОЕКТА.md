# Структура проекта (по ролям)

Код разложен по папкам — каждая роль в своей папке. Папка `Shared/` — общая основа.

> Наследование НЕ используется (по заданию не требуется). Один класс `User` с полем `Role`.

## Кто что делал (вклад участников)
- **Участник 1** — папки `Admin/` и `Customer/`
- **Участник 2** — папка `Manager/`
- **Участник 3** — папка `Chef/`
- `Shared/` — делали вместе (основа проекта)

---

## Shared/ — общая основа
| Файл | Назначение |
|---|---|
| `Program.cs` | Точка входа, запускает окно входа |
| `LoginForm.cs` | Окно входа, проверка в БД, определение роли |
| `DBHelper.cs` | Вспомогательный класс для работы с БД |
| `ThemeHelper.cs` | Цвета, шрифты, кнопки, оформление |
| `ValidationHelper.cs` | Проверка ввода (email, телефон, пароль) |
| `User.cs` | Класс пользователя (свойства, 2 конструктора, методы кошелька) |
| `ProfileForm.cs` | Профиль пользователя |
| `MenuItem.cs`, `Order.cs`, `OrderItem.cs`, `Feedback.cs`, `WalletTransaction.cs` | Классы данных |

## Admin/
| Файл | Назначение |
|---|---|
| `AdminDashboard.cs` | Главное меню админа |
